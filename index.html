Code match

仕事につながるJSスキル診断

「Code match」は、JavaScriptのコーディングスキルを可視化して、就業機会の可能性を増やすプログラムです。

このページは、Code match体験者によって、パララックスLPとして利用されます。

こんな方を応援します。

  • これからのサイト制作を一気通貫で行いたいデザイナーの方
  • JavaScriptを学んで、コーディングスキルの幅を広げたいエンジニアの方
  • JavaScriptを学んで、IOT(Internet of Things)な開発をしたいとお考えの方

Code matchの特長

就業までをトータルサポート

スキルチェックやトレーニングから、実務で活かせるお仕事紹介までを一気通貫でサポートします。

ハンズオン含め無料

現場で活躍するエンジニアの方から、JavaScriptコーディングを直に学ぶことができるハンズオン開催。参加はもちろん無料です。

就業特典でさらにトレーニング

Code match利用から就業すると、デジハリ・オンラインスクールのJavaScript講座が無償で受けられる特典をプレゼント。
